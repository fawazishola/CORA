# CORA (Core Operating & Reasoning Appliance)

**Codename:** SIERRA (Symbolic Intelligence Engine for Reasoning & Recursive Architecture)

## The Transparent Monolith
CORA is a neurosymbolic reasoning appliance built from the logic-gate level up. It rejects the generic commercial GPU paradigm in favor of vertically integrated, mathematically native hardware.

### Hardware Architecture
- **NPU Core:** Proprietary Systolic Array implemented in SystemVerilog.
- **Fabric:** Xilinx Zynq-7000 SoC (ARM PS + FPGA Logic).
- **Physicality:** 15mm optical-grade clear acrylic plates with industrial stainless-steel standoffs.
- **Aesthetic:** Matte black solder mask with exposed ENIG gold traces.

### Software Integration
- **OS:** Axiom OS (RAM-Native, Immutable).
- **Language:** [Flux](https://github.com/fawazishola/Flux) (Mathematically Native DSL).
- **Intelligence:** Alexithia (Zero-hallucination neuro-symbolic mind).

### Vision
To provide a completely air-gapped, sovereign computational artifact where hardware and intelligence are a single, verifiable entity.
